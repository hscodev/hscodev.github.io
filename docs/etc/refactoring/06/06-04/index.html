6-4 임시변수를 메서드 호출로 전환

수식의 결과를 저장하는 임시변수가 있을 때 수식을 메서드로 만들어서 참조하자. 다른 메서드에서도 사용할 수 있는 수식이 된다.

// as-is
double basePrice = _quantity * _itemPrice;
if (basePrice > 1000)
    return basePrice * 0.95;
else 
    return basePrice * 0.98;
// to-be
if(basePrice() > 1000)
    return basePrice() * 0.95;
else 
    return basePrice() * 0.98;

double basePrice(){
    return _quantity * _itemPrice;
}

동기​

임시변수는 자신이 속한 메서드의 안에서만 인식되어 임시변수에 접근하려다 보면 코드가 길어지기 마련이다. 임시변수를 메서드 호출로 전환 하면 다른 메서드에서도 접근할 수 있고 코드가 깔끔해진다.

대부분 지역변수가 많을 수록 메서드 추출이 어렵기 때문에 메서드 추출을 적용하기 전에 임시변수를 메서드 호출로 전환을 실시한다.

임시변수가 루프를 이용한 총 합계 구하기 같은 결과 누적에 사용될 때는 일부 코드를 값 반환 메서드 안으로 복사하는 작업도 필요하다.


방법​

  • 값이 한 번만 대입되는 임시변수를 찾는다.

    → 임시변수가 여러번 대입되면 임시변수 분리기법을 고려하자.

  • 임시 변수를 final로 선언한다.

  • 컴파일을 한다.

  • 대입문 우변을 빼네어 메서드로 만든다.

    → 처음엔 메서드를 private로 선언한다. 나중에 더 여러곳에서 사용하게 되면 접근 제한을 완화한다.

    → 객체 변경 등의 문제가 있다면 상태 변경 메서드와 값 반환 메서드 분리 기법을 실시하자.

  • 컴파일과 테스트를 한다.

  • 임시변수를 대상으로 임시변수 내용 직접 삽입 기법을 실시한다.

성능이 염려될 수 있다. 다른 성능 문제처럼 당장은 느려질 수 있어도 대체로 문제 없다. 문제가 된다면 최적화 과정에서 문제를 해결하면 된다.


예제​

double getPrice(){
    double basePrice = _quantity * _itemPrice;
    double discountFactor;
    if (basePrice > 1000) 
        discountFactor = 0.95;
    else 
        discountFactor = 0.98;
    return basePrice * discountFactor 
}

  • 임시 변수를 final로 선언한다.

    double getPrice(){
        final double basePrice = _quantity * _itemPrice;
        final double discountFactor;
        if (basePrice > 1000) 
            discountFactor = 0.95;
        else 
            discountFactor = 0.98;
        return basePrice * discountFactor 
    }

    컴파일시 문제가 있을 경우 경고가 출력된다. 문제가 있으면 메이 기법을 실시하지 말아야 한다.

  • 대입문 우변을 빼네어 메서드로 만든다.

    double getPrice(){
        final double basePrice = basePrice();
        final double discountFactor = discountFactor();
        return basePrice * discountFactor 
    }

    private double basePrice(){
        return _quantity * _itemPrice;
    }

    private double discountFactor(){
        if (basePrice() > 1000) 
            return 0.95;
        else 
            return 0.98;
    }

  • 임시변수 내용 직접 삽입 기법을 실시한다

    double getPrice(){
        return basePrice() * discountFactor();
    }
