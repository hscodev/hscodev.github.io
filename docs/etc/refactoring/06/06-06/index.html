6-6 임시변수 분리

루프 변수나 값 누적용 임시변수가 아닌 임시변수에 여러 번 값이 대입될 댄 각 대입마다 다른 임시변수를 사용하자.

// as-is
double temp = 2 * (_height + _width);
System.out.println(temp);
temp = _height + _width;
System.out.println(temp);
// to-be
final double perimeter = 2 * (_height + _width);
System.out.println(perimeter);
final double area = _height + _width;
System.out.println(area);

동기​

임시변수의 용도로는 누적과 간편히 참조할 수 있는 용도가 있다. 간편히 사용하는 변수에는 값이 한 번만 대입되어야 한다. 두번 이상 대입되는 경우 변수가 메서드 안에서 여러 용도로 사용된다는 반증이다. 각 용도에 맞게 분리해야한다. 코드를 분석하는 사람에게 혼동을 줄 수 있기 때문이다.


방법​

  • 선언문과 첫 번째 대입문에 있는 임시변수 이름을 변경한다.

    → 대입되는 부분에 i = i + 수식 형태라면 누적용 임시변수이므로 분리하면 안된다.

  • 이름을 바꾼 새 임시변수를 final로 선언한다.

  • 그 임시변수의 모든 참조 부분을 두 번째 대입문으로 수정한다.

  • 두 번째 대입문에 있는 임시변수를 선언한다.

  • 컴파일과 테스트를 실시한다.

  • 각 대입문마다 차례로 선언문에서 임시변수 이름을 변경하고, 그 다음 대입문까지 참조를 수정하며 위의 과정을 반복하자.


예제​

다음 예제는 돌맹이가 이동한 거리를 계산한다. 처음 가만히 있는 상태에서 첫 번째 힘을 받는다. 조금 있다가 두 번째 힘으로 이동이 가속된다. 일반적인 운동법칙을 이용하여 다음과 같이 작성할 수 있다.

double getDistanceTravelled (int time){
    double result;
    double acc = _primaryForce / _mass;
    int primaryTIme = Math.min(time, _delay);
    result = 0.5 * acc * primaryTime * primaryTime;
    int secondaryTime = time - _delay;
    if(secondaryTime > 0){
        double primaryVel = acc * _delay;
        acc = (_primaryForce + _secondaryForce) / _mass;
        result += primaryVel * secondaryTime + 0.5
            * acc * secondaryTime * secondaryTime;
    }
    return result;
}

acc가 두 번 대입되어 두가지 용도로 사용됨을 알 수 있다. 하나는 첫 번째 받은 힘으로 생긴 초기 가속도를 저장하고, 다른 하나는 두 번째 힘으로 생긴 가속을 더한 값을 저장한다.

우선 임시변수의 이름을 변경하고 그 새 변수를 final로 선언한다.

double getDistanceTravelled (int time){
    double result;
    final double primaryAcc = _primaryForce / _mass;
    int primaryTime = Math.min(time, _delay);
    result = 0.5 * primaryAcc * primaryTime * primaryTime;
    int secondaryTime = time - _delay;
    if(secondaryTime > 0){
        double primaryVel = primaryAcc * _delay;
        double acc = (_primaryForce + _secondaryForce) / _mass;
        result += primaryVel * secondaryTime + 0.5
            * acc * secondaryTime * secondaryTime;
    }
    return result;
}
